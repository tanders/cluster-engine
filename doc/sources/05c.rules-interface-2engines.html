rule-2-engines-rhythm-and-rhythm-offset-between-voices   simple-rule rhythm-engine1 rhythm-engine2 incl-rests-v1?  [Function]

Formats a rule for offset between durations (including rests) in two layers. The onsets from the 1st engine are used for inputpoints: rests and gracenotes are removed and ignored. Rests in the 2nd engine are considered valid onsets. The rule should be compiled before used. A rule receives the format '(duration-engine1 offset). Example: '(1/4 0)
